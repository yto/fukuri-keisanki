# 複利計算機

複利運用のシミュレーションができるWebアプリです。

**公開URL:** https://yto.github.io/fukuri-keisanki/

---

## 機能

- 元本・年利率・運用期間・毎月の追加積立額をスライダーで直感的に操作
- 運用結果をグラフ（積み上げ棒グラフ）と年別の表で確認
- 設定値はブラウザに自動保存（次回アクセス時に復元）
- PWA対応 — スマートフォンのホーム画面に追加してアプリとして使用可能
- オフライン動作対応（Service Worker によるキャッシュ）

## 使い方

| 項目 | 説明 |
|------|------|
| 元本 | 最初に投資する金額（¥10,000 〜 ¥100,000,000）|
| 年利率 | 年間の利回り（0.1% 〜 20%）|
| 運用期間 | 運用する年数（1年 〜 50年）|
| 追加積立 | 毎月の積み立て額（¥0 〜 ¥500,000）|

スライダーを動かすとグラフと表がリアルタイムで更新されます。

「初期値に戻す」ボタンで設定をリセットできます。

## スマートフォンへのインストール

- **iOS（Safari）:** 共有ボタン →「ホーム画面に追加」
- **Android（Chrome）:** メニュー →「ホーム画面に追加」
