# 複利計算機 仕様書

## 目的

複利運用の長期的な効果を直感的に理解できる、シンプルな Web アプリ。
スライダーを動かすだけで結果がリアルタイムに更新される。

## 対象ユーザー

- 資産運用・積立投資に興味がある個人
- 複利の効果を視覚的に確認したい人

## 提供形態

- PWA（Progressive Web App）としてブラウザで動作
- オフライン利用可能（Service Worker によるキャッシュ）
- URL: https://yto.github.io/fukuri-keisanki/

## 入力パラメーター

| パラメーター | 範囲 | ステップ | デフォルト |
|---|---|---|---|
| 元本（円） | 0 〜 1億円 | 非線形（後述） | 100万円 |
| 年利率（%） | 0 〜 30% | 0.1% | 5% |
| 運用期間（年） | 0 〜 50年 | 1年 | 10年 |
| 追加積立（円/月） | 0 〜 50万円/月 | 非線形（後述） | 2万円/月 |
| リスク・σ（年利のブレ幅） | 0 〜 30% | 0.5% | 0% |

### スライダーの目盛設計

広いレンジを直感的に操作できるよう、元本と追加積立は非線形（固定刻み配列）を採用。

**元本（PRINCIPAL_TICKS、74段階）**

| 区間 | 刻み |
|---|---|
| 0 | 0円のみ |
| 1万 〜 10万 | 5,000円刻み |
| 15万 〜 100万 | 5万円刻み |
| 150万 〜 1,000万 | 50万円刻み |
| 1,500万 〜 1億 | 500万円刻み |

**追加積立（ADDITIONAL_TICKS、37段階）**

| 区間 | 刻み |
|---|---|
| 0 | 0円のみ |
| 1,000 〜 1万 | 1,000円刻み |
| 1.5万 〜 10万 | 5,000円刻み |
| 15万 〜 50万 | 5万円刻み |

## 計算ロジック

### 基本複利計算

年ごとに以下を繰り返す：

```
balance = balance × (1 + rate) + additional_annual
invested += additional_annual
interest = balance - invested
```

- `additional_annual = 追加積立月額 × 12`
- 年初に複利を適用し、そのあと年間積立を加算する

### リスク（σ）の可視化

σ が 0 より大きいとき、「年利が毎年ランダムにブレた場合」の運用益の幅を表示する。

**モデル：対数正規分布 × 加重平均運用期間**

```
D = (元本 × y + 年間積立 × y×(y-1)/2) / 投資総額
```

- `D`：加重平均運用期間（元本は y 年フル運用、各積立はそれぞれの残り期間）

```
±1σ の運用益上限 = balance × exp(+1 × σ × √D) − invested
±1σ の運用益下限 = balance × exp(−1 × σ × √D) − invested
±2σ も同様
```

**確率的意味：**

- ±1σ 範囲に実際の利回りが収まる確率 ≈ 68%
- ±2σ 範囲に収まる確率 ≈ 95%

## 出力

### サマリーカード（4項目）

- 最終資産
- 元本累計
- 運用益累計
- 元本比（何倍になったか）

### 棒グラフ

- 積み上げ棒グラフ（Chart.js 4）
- 青：元本累計 / 緑：運用益累計
- σ > 0 のとき、各棒の上に ±1σ 箱（塗り）・±2σ ひげ（線）を重ねて描画

### 年別一覧テーブル

| 列 | 内容 |
|---|---|
| 年 | 経過年数 |
| 元本累計 | 投資した合計金額 |
| 運用益累計 | 複利で増えた分 |
| 合計資産 | 元本累計 + 運用益累計 |
| 前年比 | 1年間の増加額 |

σ > 0 のとき、運用益・合計資産・前年比それぞれに ±1σ / ±2σ の範囲を小さく表示。

## 状態管理

- スライダーの位置（インデックス or 値）を `localStorage` のキー `fukuri` に JSON で保存
- ページロード時に復元。保存データがなければ `DEFAULTS` を適用
- スライダー操作のたびにリアルタイムで保存・再計算

---

## UI / UX 方針

- スライダーを動かすだけで完結。フォーム送信やボタン操作は不要
- 「初期値」ボタン1つでリセット
- モバイルでは縦1カラム、デスクトップでは3カラムグリッドに自動切り替え
- σ の ℹ️ ボタンでツールチップ説明を表示（画面端でも切れないよう位置補正）
- グラフは Chart.js CDN から読み込み。ロード完了後に初期計算を実行
